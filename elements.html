Visualisations in
Tableau

Tableau's intuitive drag-and-drop interface allows analysts to create interactive dashboards and visualizations without extensive coding skills. You can connect to various data sources, clean and preprocess data within Tableau, and generate insights through dynamic charts, graphs, and maps.

Knowledge and skills in Tableau include:

Bins, Calculated fields, Data Connection, Data Cleaning and Transformation, Data Exploration, Basic Chart Creation, Dashboard Creation, Filtering and Sorting, Map Visualizations, Aggregation and Calculations, Parameters and Calculated Fields, Trend Analysis, Sharing and Collaboration, Data Blending, Annotations and Tooltips, Performance Optimization
